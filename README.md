# 血壓紀錄系統 (Hypertension Tracker)

這是一個現代化、響應式的血壓紀錄與追蹤系統，專為日常健康管理設計，具備功能強大的數據視覺化趨勢圖與便捷的批次操作功能。

## 核心功能

### 1. 數據分析與判定邏輯
*   **業務日期定義**：系統採用「凌晨 05:00」作為日期切換點。凌晨 05:00 以前的紀錄會被歸類為前一天的數據（與「早上」起始時間一致）。
*   **時段分類**：系統簡化為兩個時段：
    *   **早上 (Morning)**：05:00 ~ 17:59 (涵蓋日間所有紀錄)。
    *   **晚上 (Evening)**：18:00 ~ 04:59 (夜間至凌晨紀錄，跨日 05:00 以前屬前一日)。
*   **高血壓提醒**：
    *   收縮壓 (SYS) ≥ 140 mmHg 或 舒張壓 (DIA) ≥ 90 mmHg 時，數值會以**醒目紅色**顯示。
*   **最新紀錄追蹤**：表格中最頂端的最新一筆紀錄會帶有**紅色脈動邊框**。
*   **格式標準化**：日期（月、日）與時間（時、分）均採用 **兩位數補零** 顯示，報表呈現更整齊。

### 2. 趨勢圖 (Chart.js)
*   **時間範圍**：固定顯示過去 7 天的數據趨勢。
*   **平均值計算邏輯**：系統會根據選取時段自動計算平均（特別優化 722 測量規律）：
    *   **早上數據點**：取該時段內**最早的兩筆**紀錄進行平均。
    *   **晚上數據點**：取該時段內**最晚的兩筆**紀錄進行平均。
    *   **單筆處理**：若該時段內僅有一筆紀錄，則直接採用該筆數值顯示。
*   **配色規範**：
    *   🔴 **紅色 (Red)**：心跳 (HR)
    *   🟠 **橘色 (Orange)**：高血壓 (SYS)
    *   🟢 **綠色 (Green)**：低血壓 (DIA)
*   **手機優化 (Landscape Mode)**：在行動裝置上，趨勢圖會自動**旋轉 90 度**橫屏顯示。

### 3. 操作與管理
*   **修改紀錄**：點擊表格中任意紀錄即可將資料載入表單，支援即時**修改數值與更新**。
*   **批次管理**：具備「全選/取消」功能，可一次選取多個項目進行批次刪除。
*   **資料備份**：
    *   **匯出 TXT**：將所有紀錄匯出為 CSV 格式的 TXT 檔案。
    *   **匯入 TXT**：支援備份檔案重新匯入，方便跨裝置同步。
*   **資料儲存**：所有數據皆儲存於瀏覽器本地 (`localStorage`)，資料不外傳，兼顧隱私與穩定。

## 視覺設計
*   **macOS 風格**：採用 Apple 設計規範的高質感介面。
*   **玻璃擬態 (Glassmorphism)**：高斯模糊背景與半透明磨砂質感，極具現代感。
*   **動態效果**：包含平滑的平移過渡、彈性縮放按鈕，以及最新紀錄的脈動警示。
