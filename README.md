# 血壓紀錄系統 (Hypertension Tracker)

這是一個現代化、響應式的血壓紀錄與追蹤系統，專為日常健康管理設計，具備功能強大的數據視覺化趨勢圖與便捷的批次操作功能。

## 核心功能

### 1. 數據分析與判定邏輯
*   **業務日期定義**：系統採用「凌晨 03:00」作為日期切換點。凌晨 03:00 以前的紀錄會被歸類為前一天的數據。
*   **高血壓提醒**：
    *   收縮壓 (SYS) ≥ 140 mmHg 或 舒張壓 (DIA) ≥ 90 mmHg 時，數值會以**醒目紅色**顯示。
*   **最新紀錄追蹤**：表格中最頂端的最新一筆紀錄會帶有**紅色脈動邊框**，方便即時確認。

### 2. 趨勢圖 (Chart.js)
*   **時間範圍**：固定顯示過去 7 天的完整數據窗口。
*   **每日雙節點設計**：每一天固定呈現「早上」與「晚上」兩個數據點：
    *   **早上 (Morning)**：
        1. 優先取用備註為「藥前」開頭的紀錄。
        2. 若無標註且當天僅一筆：06:00 以前歸為早上，以後歸為晚上。
        3. 若無標註且當天多筆：取當天第一筆。
    *   **晚上 (Evening)**：取當天時間最晚的一筆紀錄。
*   **配色規範**：
    *   🔴 **紅色 (Red)**：心跳 (HR)
    *   🟠 **橘色 (Orange)**：高血壓 (SYS)
    *   🟢 **綠色 (Green)**：低血壓 (DIA)
*   **手機優化 (Landscape Mode)**：在手機等行動裝置上，開啟趨勢圖時會自動**旋轉 90 度**強制橫屏顯示，提供最大化的視覺空間。

### 3. 操作與管理
*   **批次刪除**：具備「全選/取消」功能，可一次選取多個項目進行批次刪除。
*   **資料備份**：
    *   **匯出 TXT**：將所有紀錄匯出為 CSV 格式的 TXT 檔案（包含 UTF-8 BOM 確保相容性）。
    *   **匯入 TXT**：支援將備份檔案重新匯入，方便跨裝置同步。
*   **資料儲存**：所有數據皆儲存於瀏覽器本地 (`localStorage`)，無須網路連接即可使用。

## 視覺設計
*   **黑暗模式**：深色的玻璃擬態 (Glassmorphism) 設計，保護眼力並提升產品質感。
*   **對齊優化**：所有操作按鈕經過高度標準化，具備對稱、美觀的佈局。
